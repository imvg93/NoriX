Troubleshooting Steps

If you still see clientReferenceManifest error:
  1. Clear Next.js cache: npm run clear-cache
  2. Delete .next folder manually
  3. Restart development server: npm run dev
  4. Hard refresh browser (Ctrl+Shift+R)
If socket connection issues persist:
  1. Check browser console for socket logs
  2. Verify backend server is running on correct port
  3. Check network tab for connection attempts
  4. Use the SocketStatus component in development mode
