Testing Instructions

To test your favicon:
  1. Create the favicon files as instructed
  2. Place them in /public/img/ directory
  3. Run npm run update-favicon
  4. Start your development server: npm run dev
  5. Open this file in your browser: http://localhost:3000/test-favicon.html
  6. Check the browser tab for your favicon
  7. Test on different devices and browsers
